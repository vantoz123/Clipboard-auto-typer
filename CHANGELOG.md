# Changelog

## v2.1.0

- Added shortcut keys:
  - Ctrl+Shift+S: Start monitoring
  - Ctrl+Shift+X: Stop monitoring and stop typing
  - Ctrl+Shift+P: Pause/resume typing
  - Ctrl+Shift+T: Type current clipboard
  - Ctrl+Shift+H: Show/hide application window
  - Ctrl+Alt+Esc: Emergency stop
  - Esc: Emergency stop while app window is focused
- Added minimize-to-system-tray behavior.
- Added tray icon menu with Show, Start, Stop, Pause/Resume, Type Current Clipboard, and Exit.
- Added saved non-sensitive settings.
- Added Safe Insert and Human Type modes.
- Added latest clipboard item queue while typing is already active.
- Added beginner scripts: install_dependencies.bat, run_app.bat, run_diagnostics.bat, build_exe.bat.
- Added diagnostics.py.
- Updated README.md and USER_MANUAL.pdf.
